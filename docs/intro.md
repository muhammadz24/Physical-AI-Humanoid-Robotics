# Welcome to Physical AI & Humanoid Robotics

**An AI-Native Interactive Textbook**

This comprehensive textbook covers the fundamentals of Physical AI, ROS 2, and Humanoid Robotics, taking you from basic concepts to advanced implementations.

## What You'll Learn

This textbook is organized into **6 comprehensive chapters**:

1. **Introduction to Physical AI** - Understanding the intersection of AI and physical systems
2. **Basics of Humanoid Robotics** - Core components and kinematics
3. **ROS 2 Fundamentals** - Robot Operating System architecture and programming
4. **Digital Twin Simulation** - Gazebo and Isaac Sim for realistic testing
5. **Vision-Language-Action Systems** - Integrating perception, language, and control
6. **Capstone Project** - Building a complete AI-Robot pipeline

## How to Use This Textbook

### Navigation
- Use the **sidebar** to navigate between chapters and sections
- Use **breadcrumbs** at the top to track your location
- Enable **dark mode** using the toggle in the navigation bar

### Interactive Features
- **Code Examples**: All code blocks include syntax highlighting, line numbers, and copy buttons
- **Self-Assessment**: Each chapter ends with questions to test your understanding
- **Further Resources**: Curated links to official documentation and tutorials

## Prerequisites

To get the most out of this textbook, you should have:
- Basic programming knowledge (Python preferred)
- Understanding of fundamental mathematics (linear algebra basics)
- Access to Ubuntu 22.04 (for running ROS 2 examples)

## Get Started

Ready to begin? Head to [Chapter 1: Introduction to Physical AI](./chapter-01/index.md) to start your learning journey!
